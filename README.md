 🔍 Overview
This project presents an interactive **Sales Performance Dashboard** built using Microsoft Excel. The dashboard provides insights into sales trends, customer behavior, product performance, and channel contribution.

To maintain data confidentiality, the dataset used in this project is a **synthetically generated dataset** created to mimic real-world sales scenarios.

🎯 Objectives
- Analyze overall sales performance
- Track monthly and quarterly trends
- Identify top-performing products and categories
- Understand customer-level metrics
- Compare sales across different channels and outlet types

📁 Dataset
- The dataset is **dummy/simulated data**
- Designed based on real business structure
- Fields included:
  - Date
  - Sales Value
  - Volume
  - Customer ID
  - Sub Category
  - Channel Type
  - Outlet Type

🛠 Tools & Techniques Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers (Interactive Filters)
- Data Cleaning
- Data Modeling (Basic)
- KPI Design

📊 Key Features
🔹 KPI Metrics
- 💰 Total Sales
- 📦 Total Volume
- 📈 Average Sales per Day
- 💳 Sales per Customer

🔹 Visualizations
- 📈 Monthly Growth Trend
- 📊 Quarter-wise Sales
- 📊 Sub-category Performance
- 🏆 Top Selling Products
- 🥧 Channel Contribution (Retail vs Wholesale)

🔹 Interactivity
- Slicers for:
  - Month
  - Sub Category
  - Channel Type
  - Outlet Type

🧠 Key Insights
- Q4 recorded the highest sales performance
- Biscuits emerged as the top-performing sub-category
- Wholesale channel slightly outperformed retail
- Certain products like premium biscuits contributed significantly to revenue
- Average revenue per customer indicates strong customer value

⚠️ Note
This project uses **synthetic data** for demonstration purposes. It does not contain any real company data.
